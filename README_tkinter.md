# GPT-SoVITS Tkinter界面

这是GPT-SoVITS文本转语音系统的Tkinter界面版本，提供了一个更轻量的GUI选择，功能与原Gradio界面类似。

## 功能特点

- 使用Tkinter实现，无需安装额外的Web框架
- 与原版保持一致的文本处理和语音合成能力
- 支持角色管理和配置
- 多种文本处理模式：逐行处理、全文本处理和分段处理

## 使用方法

### 运行界面

```bash
python tkinter_app.py
```

### 文本文件标签页

1. **处理模式**：
   - **逐行处理**：每行文本单独处理，支持行内角色和情绪标记
   - **全文本处理**：使用默认角色和情绪处理整个文本，适用于大段独白
   - **分段处理**：将文本分成多个段落(不超过500字)，分别处理后合并，解决长文本问题

2. **文本输入**：
   - 直接在文本框中输入文本
   - 或点击"上传文本文件"按钮选择文本文件

3. **角色和情绪设置**：
   - **默认角色**：当文本没有指定角色时使用的角色（必选）
   - **默认情绪**：当文本没有指定情绪时使用的情绪
   - **强制角色/情绪**：忽略文本中的角色/情绪标记，全部使用指定值

4. **文本格式**：
   - 普通文本：直接输入，使用默认角色和情绪
   - 标记角色：`(角色名)文本内容`
   - 标记角色和情绪：`(角色名|情绪)文本内容`

5. **预处理文本**：
   - 将双引号内的文本作为对白（使用默认情绪）
   - 其他文本作为叙述

### 角色管理标签页

1. **角色列表**：
   - 查看已配置的所有角色
   - 点击"加载角色"按钮加载选中的角色配置和模型
   - 点击"删除角色"按钮删除选中的角色
   - 点击"刷新角色列表"按钮更新角色列表
   - 点击"保存当前角色"按钮将当前设置保存为角色配置

2. **模型选择**：
   - 选择要使用的GPT和SoVITS模型
   - 点击"刷新模型列表"按钮更新可用模型

3. **参考音频**：
   - 设置参考音频文件路径（点击"浏览..."按钮选择音频文件）
   - 输入参考音频的文本内容
   - 设置相关语言参数
   - 可选择开启"无参考文本模式"

4. **情绪设置**：
   - 添加、删除情绪
   - 为每个情绪设置说话风格(s1)、情感程度(s2)和音高(s3)参数
   - 查看当前角色的情绪列表

5. **合成设置**：
   - 输入需要合成的文本
   - 设置断句符号、语速、停顿秒数等参数
   - 调整采样步数（影响合成质量和速度）
   - 可选择启用超采样提高语音质量

6. **操作按钮**：
   - "创建新角色"：创建并保存新角色配置
   - "测试合成"：使用当前设置测试合成语音
   - "播放"：播放合成的音频

### 配置替换音标签页

在这里可以编辑词语替换配置，格式为：
```
替换前 替换后
例如：
test 测试
```

词语替换可以在合成过程中自动将特定词语替换为其他词语，适用于修正发音或替换不适合朗读的内容。

## 新增功能说明

相比原版Gradio界面，Tkinter版本新增了以下功能：

1. **角色列表管理**：直观的角色列表展示和管理
2. **情绪管理**：支持添加、删除和调整情绪参数
3. **批量操作**：方便地批量加载和管理角色

## 注意事项

- 此界面依赖原GPT-SoVITS的后端API，确保API服务已启动
- 配置和角色文件与原版共享，使用同一套数据
- 参考音频应控制在3~10秒之间，超过可能导致错误
- 创建新角色时，必须指定有效的参考音频文件和模型

## 系统要求

- Python 3.8+
- tkinter (通常包含在Python标准库中)
- 与GPT-SoVITS相同的依赖项
